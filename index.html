Time to Hex

This was my first attempt at converting the current time to a hex value. The background color of this window is basically the current time using the 24 hour clock. Since the (r)ed value doesn't go higher than 24, and the (g)reen & (b)lue values only reach a maximum of 60, the color are very dark and on the cool side of the spectrum. Not great for display.

Time to RGBA

Here's my second attempt to convert the current time to a color. Learning from my Time to Hex experiment, I decided that I wanted to try to use the majority of the RGB color space, so I'm taking the ratio of current time by 23:59:59 and multiplying that by 255 to get a 'prettier' color. I like this version better. You might notice certain aspects of the site change color periodically. This isn't great for JavaScript paint times, I think it's a neat touch.

Random Number to Hex

This site was a mash-up of a couple ideas I had when I had some free time. The first idea I had was that I could take a random number, and using a small JavaScript library I found on the web, convert that number into a MD5 hash. Since MD5 hashes are all hexadecimal values, I then took the first 6 alphanumerics in that result, and made that the hex color of an element on the page. Not stopping there, I filled the window with these elements, and made the background. The second idea started as an inside joke at Craftsy.com. Since I was in the UK during the interview process, I soon became 'British Clint.' So I started a Twitter account, and pulled in the last tweet and displayed it on the page. Since then though, Twitter has moved to its OAuth model, so the script I was using now fails (gracefully).
